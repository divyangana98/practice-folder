Learn to use git

Steps to create a new repository, commit new changes and create PR

  1. 1. Go to github desktop appliation. go to File option in the menu bar and click on 'New Repository.
  2. 2. Enter the name of the repository and click on OK.
  3. 3. Now open visual studio. Go to file and select 'Open Folder'
  4. 4. Select the folder where you want to add your files.
  5. 5. Create a new file. The first icon with plus sign last one from left.
  6. Save changes and you can commit your changes from git. Yay!!!!
